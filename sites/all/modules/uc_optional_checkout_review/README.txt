Overview:
--------

Makes the review button on the Ubercart checkout page optional (set by store admin).

Once the module is enabled, you must go to admin/store/settings/checkout/edit/basic
to enable the "Skip checkout review" option.

Requires:
--------

 - Drupal 7.x
 - Ubercart 3.x
